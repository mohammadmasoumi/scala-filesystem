# scala-filesystem

This is an open-source **demo filesystem** project which is written in **scala 2.13.4**. 
The main purpose of this project is to practice what I've learned. 


## supported commands

 - [**Cat**][2]


### Environment
 - JDK 1.8.0
 - Scala 2.13.4
